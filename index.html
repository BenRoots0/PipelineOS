Quick Start:
1. Open or create a track in the Session Modal
2. Paste lyrics in Lyric Input (use [Section] tags)
3. Add an overall style prompt
4. Lock the input and Send ↓ to pipeline
5. Run modules in order: Wizard → Prompt Wizard → Hit Detective
6. Export via Suno Export when ready
